# Скрипт для миграции бд #


## Установка зависимостей

pip install -r requirements.txt


## Переменные окружения, необходимые для работы приложения

* Переменные для подключения к базе данных:
    - ${DATASOURCE_URL} - адрес подключения к БД
    - ${DATASOURCE_PORT} - порт подключения к БД
    - ${DATASOURCE_USERNAME} - имя пользователя для подключения к БД
    - ${DATASOURCE_PASSWORD} - пароль для подключения к БД


## Запуск приложения

* Задать переменные окружения (можно задать в системе, можно создав `.env`)
* Запустить main.py
